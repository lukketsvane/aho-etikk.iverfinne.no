# Søkelogg

Logg kvart søk her for reproduserbarheit (jf. metodikk pkt. 5). Ein rad per søk.
Kopier siste rad og fyll inn på nytt. Hald formatet stabilt så loggen kan
parsast seinare.

| Dato/tid | Søkestreng | Kjelde / API | Endepunkt / URL | Parameter | Treff | Feilkjelder / atterhald | Vidare steg |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 2026-06-28 | (døme) «lov om universiteter og høyskoler» | Lovdata-indeks | `data/index/lover.csv` | grep | 1 | korttittel kan variere | slå opp gjeldande versjon på lovdata.no |
|  |  |  |  |  |  |  |  |

## Konvensjonar

- **Dato/tid**: ISO 8601 (`2026-06-28` eller `2026-06-28T14:30`).
- **Treff**: tal, eller «0» — men skriv aldri «ingen funn» før alternative
  stavemåtar, eldre rettskriving, initialar og historiske namn er prøvde.
- **Feilkjelder**: OCR, ufullstendig dekning, tilgangssperre, tvitydig namn.
- Lim inn det eksakte endepunktet/URL-en slik at søket kan køyrast på nytt.
