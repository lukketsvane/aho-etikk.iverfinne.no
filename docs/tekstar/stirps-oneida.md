# stirps (ONEIDA)

Litterær/essayistisk tekst. Original: [`stirps-oneida.pdf`](stirps-oneida.pdf).

Teksten handlar om **Oneida-kommunen** (USA, grunnlagt 1848) - eit utopisk
samfunn med ein leiar «over ringen ingen, over lyset ingenting», som praktiserte
*stirpiculture* (planlagd avl, «stamme/rot/ætt» av lat. *stirps*), og som
seinare vart bestikk-produsenten **Oneida** (gaffelen i skuffa).

## Hvorfor i dette repoet

Tematisk parallell til AHO-saka: ein autoritet som **ikkje kan visast feil**,
utan noko «nei» over seg - same strukturelle poeng som kjernekritikken
(sjølvakkreditering, ingen ekstern instans som fangar brotet, «krinsen som skulle
fange brotet er huset sjølv»). Teksten er **kontekst/ramme**, ikkje bevis, og
inngår ikkje i funn- eller påstandsregisteret.
