# Funnlogg

Speglar Notion-databasen **«AHOs lumskheit»** (Funn: regelbrot og løftebrot).
Flytt rader 1:1 mellom denne fila og Notion. Felt = same skjema.

- **Type**: Regelbrot · Løftebrot · Ansvarsglipe · Faktasjekk · Styringssvikt ·
  Interessekonflikt · Historisk forelegg · Alumni-fagleg · Alumni-personleg
- **Styrke**: Sterk · Medium · Svak
- Eit funn er ikkje «funn» før **Regel/lovnad → Røyndom/brot → Kjelde** er fylt.
