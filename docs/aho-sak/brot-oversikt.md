# Brot og løftebrot - samla, rangert oversikt

Konsolidering av **alt** i Notion-funndatabasen «AHOs lumskheit» (206 kjeldeførte
rader) + GitHub-bevismatrisa (runde 1-3). Rangert etter **kor klart/sannsynleg
funnet er** dersom saka må eskalerast (NOKUT, KD, Riksrevisjonen, Sivilombodet,
Datatilsynet, Språkrådet, Arbeidstilsynet, eller offentleg publisering).

> **Bindande atterhald.** Dette er ein arbeidsmatrise over **funn som må
> verifiserast**, ikkje fastslått skuld. Før eskalering/publisering: gi AHO og
> namngjevne personar **samtidig imøtegåing** (Ver Varsam 4.14). Skil
> **institusjonen AHO** frå **enkeltpersonar/alumni** (sistnemnde er ikkje AHOs
> ansvar). Sensitive geopolitiske koplingar (Kongsberg/F-35/Israel, Palantir)
> krev streng, fleirkjelde-dokumentasjon og skal ikkje overdrivast.

Styrke: 🟢 Sterk (primærkjeldefesta, toler tilsvar) · 🟡 Medium (krev litt meir) ·
🔴 Svak/indikasjon (treng innsyn/verifisering).

## Tematiske klynger (for ein eventuell sak)

1. **Etikk-infrastrukturen manglar** (kjernen): #1, #2, #9, #10, #17, #20, #21 +
   benchmark (AHO blant svakaste norske). → **NOKUT/KD/offentleg.**
2. **Forsvars-/overvakingsteknologi** (mest konsekvensrik, mest sensitiv):
   #7, #8, #19 + Kongsberg/F-35/Israel-kontekst. → **innsyn + offentleg**, streng
   kjeldekritikk.
3. **Styring og økonomi** (lengst dokumentasjonsspor): #5, #6, #11. → **Riksrevisjonen.**
4. **Lovbrot** (konkrete, etterprøvbare): #3 (NOKUT), #4 (språklov), #18
   (Datatilsynet). → respektive tilsyn.
5. **Læringsmiljø/arbeidsmiljø**: #11, #12, #24, #25, #27. → **Arbeidstilsynet/offentleg.**
6. **Løftebrot (retorikk vs. praksis)**: #14, #15, #16, #22. → **offentleg/presse.**
