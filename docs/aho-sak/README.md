# AHO-saka — granskingsdossier

Kjeldekritisk dokumentasjon av etiske og juridiske **tillitsbrot** og
**løftebrot** ved **Arkitektur- og designhøgskolen i Oslo (AHO)**. Dette er
arbeidsstrukturen i git; den løpande granskinga skjer i Notion (sjå
[`../notion.md`](../notion.md)).

## Kjernepåstand (frå Notion «brev til killi»)

> AHO lærer designstudentar å be om å sjå overvakingssystem, men ikkje å seie nei
> til dei. Skulen har ikkje noko etisk organ, ingen bindande retningsliner og
> ingen instans som kan stoppe eit prosjekt. Etikk og berekraft vert lærte som
> tillegg, ikkje som krav. Eit lovpålagt læringsutbyte om etikk står i
> studieplanen, men vert aldri prøvd eller vurdert.

Krav: ein designskule må ha eit **«nei»** — offentleg etisk vurdering av eksterne
samarbeid, rett til å reservere seg, og obligatorisk vurdering av klima, miljø og
makt i alle større prosjekt.

Dette er **hypotesen som skal provast eller avkreftast** — ikkje ein konklusjon.

## Omgrep

- **Løftebrot** — skulen lova/marknadsførte noko (studieplan, læringsutbyte,
  strategi, verdiar) som ikkje vert innfridd i praksis.
- **Tillitsbrot / regelbrot** — brot på lov, forskrift, eigne reglar eller
  forvaltningsskikk (offentleglova, forvaltningslova, UH-lova, økonomi-/
  anskaffingsregelverk).
- **Ansvarsglipe / styringssvikt** — manglande organ, prosedyre eller
  ansvarsplassering som gjer brot mogleg.
- **Interessekonflikt** — bindingar mellom AHO-roller og eksterne interesser.

(Same typologi som `Type`-feltet i Notion-funndatabasen.)

## Ramme — bindande

- **Nøytralt og bevisbasert.** Ingen førehandsdom. Kvar påstand: *regel/lovnad →
  røyndom/brot → kjelde → atterhald*, gradert Sterk/Medium/Svak.
- **Samtidig imøtegåing.** AHO og namngjevne personar i offentleg rolle skal få
  tilsvar før publisering (Ver Varsam 4.14).
- **Personvern.** Personregister blir i Notion (tilgangsstyrt), ikkje i git. Skil
  offentleg journalmetadata frå dokument; be om innsyn.
